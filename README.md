# Hoax Call Detection Project

A comprehensive machine learning pipeline for clustering fraudulent/hoax phone calls using advanced NLP techniques, clustering analysis, and multiple classification algorithms.

## Setup Instructions

1. Clone the repository:

   ```
   git clone <repository-url>
   cd hoax-call-detection
   ```

2. Install the required dependencies:

   ```
   pip install -r requirements.txt
   ```

## Usage

To run the hoax call detection algorithm, execute the following command:

```
python src/main.py
```

This will load the dataset, preprocess the data, apply clustering algorithms, and evaluate the results.
